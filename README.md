# Simple SEO Audit Tool

A simple tool to check basic SEO elements (Title, Meta Description, H1, OG Tags, Canonical) from any URL.

## Tech Stack
- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Parser:** Cheerio

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000)

## Features
- ✅ Basic SEO element extraction
- ✅ Clean and responsive UI
- ✅ Web3-ready architecture (ready for wallet integration)
